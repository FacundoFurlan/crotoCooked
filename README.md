# Pollo, Bifes y Achicoria

Pollo, Bifes y Achicoria es un juego de cocina donde debemos gestionar la cocina de un carribar de parrillada precario compitiendo contra nuestro compañero de trabajo para evitar ser despedido por recorte de personal.

Además se sabe que por esta región asechan peligrosas criaturas que podrían poner en peligro el carribar...

o... salvarlo de la quiebra?
